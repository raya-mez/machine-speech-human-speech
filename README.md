# Survey building

## Stimulus creation and assignment
1. Ten essay topics are selected among those used by Herbold et al. (2023). Each topic is assigned a unique ID. 
2. Essays on the selected topics are generated by ChatGPT 5.2 through the web interface using the prompt: 
    >Write an essay of about 200 words on "[topic]"
3. Each obtained text is assigned as stimulus to one of the 10 chains (indexed A-J). Each chain is also assigned three other texts as primes (by referring to the chains for which they are used as stimuli). The data for each chain is stored in `stimuli/gen1/herbold23_based/chain_data.json` and includes the stimulus title, text, word count, and list of sentences, and a the indices of the three chains whose stimuli are used as primes. 


## Survey versioning
1. A starter template is created through the Qualtrics web interface. It contains placeholder main task blocks for each chain with the appropriate branching logic. 
2. The survey template is downloaded as a .qsf file, which follows a .json format. Thus, a .json copy is created and used to programmatically create the survey versions by injecting generation- and chain-specific data in `scripts/survey_versioning/create_surveys.py`. This script uses local copies of the HTML and Javascript template for easier (better IDE) and safer (automatic saving, not relying on internet) development and version control.
